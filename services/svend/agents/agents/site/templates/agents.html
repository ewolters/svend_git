Workflow Engine

Chain agents together. Save as Python scripts. Re-run anytime. 

from workflow import Workflow, ResearchStep, CoderStep, WriterStep

workflow = Workflow("My Analysis")
workflow.add(ResearchStep("quantum computing basics"))
workflow.add(CoderStep("simulate qubit", uses=["research"]))
workflow.add(WriterStep("technical_spec", uses=["research", "code"]))

result = workflow.run()
print(result.to_markdown())

# Save as reusable Python script
workflow.save("my_workflow.py")

Available Steps

CoderStep - Generate code

ExperimentStep - Design experiments

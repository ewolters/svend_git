How to build a VSM

1. Drag Process boxes onto the canvas.

2. Double-click to enter metrics: C/T, C/O, Uptime, operators.

3. Add Inventory triangles between steps (days of supply).

4. Connect with Push or Pull flow arrows.

5. Set Takt Time to see bottlenecks.

6. Add Kaizen bursts for improvement opportunities.

5S → FMEA Cross-feed rule: When a 5S deficiency carries safety consequence (e.g., missing LOTO shadow board = 5S-SET failure + energy hazard), it enters the FMEA as a hazard. The 5S Pareto tracks frequency for standardization; the FMEA tracks risk for control. Same observation, two governance paths.
